# Default Files

In your application there are a few default files:

- default.html
- package.appxmanifest
- [css]
    - default.css
- [images]
    - logo.png
    - smalllogo.png
    - splashscreen.png
    - storelogo.png
- [js]
    - default.js

### 1. default.html

This is the main view/layout file of your app.
