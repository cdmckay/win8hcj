## Support Programs

Microsoft programs to help you build apps as a freelance developer, student or a startup/small company.  These programs give you access to free legit Microsoft software, like full blown Visual Studio, Windows 8 and other software to help you build apps. 

1. [The Developer Movement](https://www.developermovement.ca/default.aspx)
2. [DreamSpark (for students/educational institutions)](https://www.dreamspark.com/)
3. [BizSpark (for small businesses/startups aka just businesses)](http://www.microsoft.com/bizspark/)
